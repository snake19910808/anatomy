Red Flags — 禁忌症守則（先讀）

以下情況絕對禁止進行湯普森手法調整：

🩸 腹主動脈瘤
腹部中線大血管，位於 L4 前方，觸診時若感到搏動性腫塊立即停止
🦴 嚴重骨質疏鬆 / 骨折
下落衝擊力可能造成椎體壓迫性骨折
🔴 脊椎腫瘤 / 感染
不明原因夜間疼痛 + 體重驟降 = 紅旗，先轉介影像
⚡ 馬尾症候群
雙腿麻木 + 二便失控 = 神經急症，立即送醫
🩺 嚴重脊椎滑脫（Grade III+）
椎體位移超過 50%，Drop Table 衝擊有加重風險
🩺 急性椎間盤突出（神經症狀期）
放射痛 + 肌力喪失期間，避免直接脊椎調整

湯普森共通語言 — 方向向量完整說明

湯普森手法用一套標準化的方向代碼來描述「在哪裡施力、往哪個方向推」。這是所有調整的共通語言，必須先記住才能理解後面的錯位分類。

六個基本方向（解剖學座標）
P → A
Posterior → Anterior
由後往前
A → P
Anterior → Posterior
由前往後（少用）
I → S
Inferior → Superior
由下往上
S → I
Superior → Inferior
由上往下
M → L
Medial → Lateral
由內側往外側
L → M
Lateral → Medial
由外側往內側
記憶口訣： P＝背（Posterior=背部方向）、A＝肚（Anterior=腹部方向）、I＝腳（Inferior=腳的方向）、S＝頭（Superior=頭頂方向）、M＝中線（Medial=靠近身體中線）、L＝外側（Lateral=遠離中線）
湯普森手法全部調整向量總表
錯位名稱 意思 接觸點 向量（施力方向）
PI-IN 後下＋內旋 PSIS 內側緣 P-A ＋ I-S ＋ M-L
PI-EX 後下＋外旋 PSIS 外側緣 P-A ＋ I-S ＋ L-M
AS-IN 前上＋內旋 坐骨結節 內側緣 P-A ＋ S-I ＋ M-L
AS-EX 前上＋外旋 坐骨結節 外側緣 P-A ＋ S-I ＋ L-M
薦椎前傾 (SP-Flex) 薦椎底往前傾 薦椎底 P-A ＋ I-S
薦椎後傾 (SP-Ext) 薦椎底往後仰 薦椎底 P-A ＋ S-I
C1 旋轉錯位 環椎左/右旋 C1 橫突 P-A ＋ M-L（朝對側）
C2 棘突錯位 軸椎旋轉 C2 棘突 P-A ＋ 視旋轉方向
三個字母向量的閱讀方式：
以 P-A ＋ I-S ＋ M-L 為例：
→ 同時往「前」+ 往「上」+ 往「外側」施力
→ 這三個方向不是分開做，而是合成一個斜角力向量，一次施力完成。
🧠

概念導入：翹翹板 + 開合門

PI / AS = 翹翹板（垂直面）
PI（Posterior Inferior）：PSIS 往下，ASIS 往上
AS（Anterior Superior）：PSIS 往上，ASIS 往下
口訣：PI 找後下 ▼，AS 找前上 ▲
IN / EX = 開合門（水平面）
IN（Internal）：髂骨往內關攏（靠近薦椎）
EX（External）：髂骨往外展開（遠離薦椎）
口訣：IN 向內，EX 向外
5 大接觸點口訣：
🔴 PSIS → PI 髂骨調整點 🔴 坐骨結節 → AS 髂骨調整點 🔴 薦椎底 → 薦椎前/後傾調整點 🔴 C1 橫突 → 環椎旋轉調整點 🔴 C2 棘突 → 軸椎錯位調整點
