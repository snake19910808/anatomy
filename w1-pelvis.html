Red Flags — 禁忌症守則（先讀）

以下情況絕對禁止進行湯普森手法調整：

🩸 腹主動脈瘤
腹部中線大血管，位於 L4 前方，觸診時若感到搏動性腫塊立即停止
🦴 嚴重骨質疏鬆 / 骨折
下落衝擊力可能造成椎體壓迫性骨折
🔴 脊椎腫瘤 / 感染
不明原因夜間疼痛 + 體重驟降 = 紅旗，先轉介影像
⚡ 馬尾症候群
雙腿麻木 + 二便失控 = 神經急症，立即送醫
🩺 嚴重脊椎滑脫（Grade III+）
椎體位移超過 50%，Drop Table 衝擊有加重風險
🩺 急性椎間盤突出（神經症狀期）
放射痛 + 肌力喪失期間，避免直接脊椎調整

湯普森共通語言 — 方向向量完整說明

湯普森手法用一套標準化的方向代碼來描述「在哪裡施力、往哪個方向推」。這是所有調整的共通語言，必須先記住才能理解後面的錯位分類。

六個基本方向（解剖學座標）
P → A
Posterior → Anterior
由後往前
A → P
Anterior → Posterior
由前往後（少用）
I → S
Inferior → Superior
由下往上
S → I
Superior → Inferior
由上往下
M → L
Medial → Lateral
由內側往外側
L → M
Lateral → Medial
由外側往內側
記憶口訣： P＝背（Posterior=背部方向）、A＝肚（Anterior=腹部方向）、I＝腳（Inferior=腳的方向）、S＝頭（Superior=頭頂方向）、M＝中線（Medial=靠近身體中線）、L＝外側（Lateral=遠離中線）
湯普森手法全部調整向量總表
錯位名稱 意思 接觸點 向量（施力方向）
PI-IN 後下＋內旋 PSIS 內側緣 P-A ＋ I-S ＋ M-L
PI-EX 後下＋外旋 PSIS 外側緣 P-A ＋ I-S ＋ L-M
AS-IN 前上＋內旋 坐骨結節 內側緣 P-A ＋ S-I ＋ M-L
AS-EX 前上＋外旋 坐骨結節 外側緣 P-A ＋ S-I ＋ L-M
薦椎前傾 (SP-Flex) 薦椎底往前傾 薦椎底 P-A ＋ I-S
薦椎後傾 (SP-Ext) 薦椎底往後仰 薦椎底 P-A ＋ S-I
C1 旋轉錯位 環椎左/右旋 C1 橫突 P-A ＋ M-L（朝對側）
C2 棘突錯位 軸椎旋轉 C2 棘突 P-A ＋ 視旋轉方向
三個字母向量的閱讀方式：
以 P-A ＋ I-S ＋ M-L 為例：
→ 同時往「前」+ 往「上」+ 往「外側」施力
→ 這三個方向不是分開做，而是合成一個斜角力向量，一次施力完成。
🧠

概念導入：翹翹板 + 開合門

PI / AS = 翹翹板（垂直面）
PI（Posterior Inferior）：PSIS 往下，ASIS 往上
AS（Anterior Superior）：PSIS 往上，ASIS 往下
口訣：PI 找後下 ▼，AS 找前上 ▲
IN / EX = 開合門（水平面）
IN（Internal）：髂骨往內關攏（靠近薦椎）
EX（External）：髂骨往外展開（遠離薦椎）
口訣：IN 向內，EX 向外
5 大接觸點口訣：
🔴 PSIS → PI 髂骨調整點 🔴 坐骨結節 → AS 髂骨調整點 🔴 薦椎底 → 薦椎前/後傾調整點 🔴 C1 橫突 → 環椎旋轉調整點 🔴 C2 棘突 → 軸椎錯位調整點

功能性力學 — 吊車掛鉤比喻

為什麼 PI 髂骨錯位會造成「短腿」？理解這條因果鏈，Derifield 測試就完全說得通。

三步因果鏈
①
髂骨向後下旋轉（PI）
PSIS 往下、ASIS 往上。整塊髂骨繞薦髂關節做「後下旋轉」。
↓
②
髖臼跟著上移
髖臼（Acetabulum）是髂骨的一部分，髂骨後下旋轉時髖臼相對上提。
💡 把髖臼想成「吊車的掛鉤」— 吊臂旋轉，掛鉤跟著動
↓
③
股骨被拉高 → 功能性短腿
股骨頭坐在髖臼裡，掛鉤上移 → 股骨（腿）跟著被拉高。俯臥看腳跟，該側「看起來」較短。
左：正常　右：PI 錯位
✅ 功能性短腿（Thompson範疇）
骨骼長度實際相同，是骨盆旋轉改變股骨「起點」。矯正骨盆後腿長立刻恢復。
📏 結構性短腿（非Thompson範疇）
骨骼本身長度不等，需X光確認，無法透過脊椎調整改善。
Derifield 的力學邏輯：
彎膝90度後骨盆肌肉張力改變，旋轉代償被抵消。短腿消失（Balance）= 肌肉問題；短腿維持（Stay Short）= 骨盆旋轉本身 = PI 髂骨錯位。
口訣：
後下旋轉 → 髖部上提 → 腿部縮短
PI = Posterior Inferior 後下　→　Acetabulum rises 髖臼上移　→　Leg appears short 腿縮短

視覺辨識：IN/EX 閉孔對比 + X光三基線

IN vs EX — 閉孔（Obturator Foramen）形狀變化

髂骨旋轉會改變閉孔的外觀形狀，這是X光判讀的重要線索：

IN（內旋）
髂骨內旋 → 骨盆往中收攏 → 閉孔被「擠」成高窄橢圓
EX（外旋）
髂骨外旋 → 骨盆往外張開 → 閉孔被「撐」成扁圓形
記憶法： IN = 往內 = 閉孔往內縮 = 窄橢圓。EX = 往外 = 閉孔往外撐 = 寬圓。
X光骨盆判讀 — 三條基線

湯普森評估時，X光片上需要畫出三條水平基線來判讀骨盆對稱性：

① 髂嵴高度連線 連接兩側髂嵴最高點。不等高 → 骨盆側傾，可能有 PI 或 AS 錯位。
② 薦椎底水平線 沿薦椎底（S1上緣）畫橫線。傾斜 → 薦椎側傾，影響 Flex/Ext 判讀。
③ 坐骨水平線 連接兩側坐骨最低點。不等高 → 骨盆旋轉；PI 側坐骨通常較高（向上旋轉）。

觸診 SOP + 臨床日誌

標準化的觸診流程，確保每次評估一致、不遺漏。建議每次練習後填寫日誌。

6步觸診SOP
S1
體位確認：患者俯臥，手臂放鬆置於身體兩側，雙腳自然伸直（不要交叉）。治療師站在床側。
S2
初步腿長評估：從床尾觀察兩側內踝（或足跟）高度差。記錄哪側較短。
S3
PSIS 觸診（兩側）：從臀部中線旁往上滑動，找到「酒窩」位置的骨突。雙拇指分別置於左右 PSIS，比較高低。
S4
髂嵴高度評估：雙手掌沿髂嵴從後往前滑動，感受兩側高度差。
S5
Derifield 測試：雙手輕握踝關節，請患者彎曲雙膝90度。再次觀察腳跟高度。記錄 Stay Short / Balance / 等長。
S6
坐骨結節確認（若懷疑AS）：若 PSIS 高於對側（可能 AS），需觸診坐骨結節，確認調整接觸點。坐骨結節位於臀部正下方，較深層。
